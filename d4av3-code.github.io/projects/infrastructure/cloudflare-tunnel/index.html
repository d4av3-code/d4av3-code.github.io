The Concept

Zero Inbound Ports

Exposing a home server traditionally requires opening ports on a router, creating DDNS rules, and constantly monitoring firewall logs. I wanted to bypass that entirely.

This container runs the cloudflared daemon. Instead of waiting for incoming traffic, it reaches out to Cloudflare's global edge network and establishes an encrypted tunnel. When a user requests my portfolio, Cloudflare routes the traffic down this existing tunnel directly to this LXC, which then forwards it to the Web Server LXC.

Security

Invisible to the Internet

Because there are no open ports, scanners and bots crawling the internet cannot find my home IP address. All traffic is forced through Cloudflare's WAF (Web Application Firewall) before it ever touches my hardware.
