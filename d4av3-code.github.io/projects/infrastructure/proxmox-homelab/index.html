The Core

One host, many isolated containers

Instead of stacking everything onto a single operating system, a Proxmox server divides the workload into lightweight LXCs (Linux Containers). This approach keeps the environment clean and modular.

Architecture

Separation of concerns

A server should only do the job it was assigned to do. This environment is split into dedicated, single-purpose containers:
