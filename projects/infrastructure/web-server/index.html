The Stack

Lightweight & Focused

This LXC contains standard web server daemon (like Nginx or Apache) and PHP. Because the portfolio is primarily a Vanilla JS Single Page Application (SPA) that acts as its own API via JSON files, the server overhead is practically zero.

Isolation

Completely Air-gapped from the Internet

This container has no idea the outside world exists. It only listens for local traffic coming from the internal Proxmox virtual bridge. It accepts requests handed to it by the Cloudflare Tunnel LXC, processes them, and hands the static files back.
